# Shlok-Superpad
A macropad I fully designed with help from hackclub with 9 keys in a matrix, a rotary encoder, oled screen, and rgb backlighing for each key.

# CAD
I'm screwing direcrly into the plastic with 2.9mm holes for construction of the case. The 2 parts are the top (with the mercedes logo) and the bottom.
